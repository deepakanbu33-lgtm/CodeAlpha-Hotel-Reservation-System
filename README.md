# Hotel Reservation System - Java

This project is created as part of the CodeAlpha Java Programming Internship.

## Project Description
A console-based hotel reservation system that allows users to view rooms,
book rooms, and cancel reservations.

## Features
- View available rooms
- Book hotel rooms
- Cancel bookings
- Room categorization (Standard, Deluxe, Suite)
- Uses OOP and ArrayList

## Technologies Used
- Java
- OOP Concepts
- Collections (ArrayList)

## How to Run
1. Compile the program:
   javac task4.java
2. Run the program:
   java task4
